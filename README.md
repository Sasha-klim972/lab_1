## Варіант завдання (№5108)

Параметри роботи визначено за залишком від ділення номера залікової книжки:

1. **C5 (Операція):** 5108 % 5 = 3
    * Операція: Поелементне виключне "або" (XOR) матриць A та B.
2. **C7 (Тип даних):** 5108 % 7 = 5
    * Тип елементів матриці: `char` (символьний тип).
3. **C11 (Дія з матрицею):** 5108 % 11 = 4
    * Завдання: Обчислити суму найбільших елементів у рядках з парними номерами (0, 2, 4...) та найменших елементів у рядках з непарними номерами (1, 3, 5...).

## Опис реалізації

Проєкт реалізовано на мові Java з використанням системи збірки **Maven**. Основна логіка розподілена наступним чином:

* **Matrix.java**: Клас, що описує об'єкт "Матриця". Містить методи для виконання операції XOR та розрахунку підсумкової суми згідно з варіантом. Використовує принципи інкапсуляції.
* **Lab1.java**: Головний клас для запуску програми, ініціалізації даних та виводу результатів у консоль.
* **AppTest.java**: Модульні тести (JUnit 5) для автоматичної перевірки коректності обчислень.
